Class Proxy

A read-only proxy pointing to a decoded JSON result table. Proxies are regular empty tables with their __index, __pairs and __ipairs metamethods pointing to the decoded result table. Access to a subtable of the result table will return a Proxy pointing to the subtable. Non-table values are returned as normal. Attempting to change the value of a Proxy's key will throw an error.
